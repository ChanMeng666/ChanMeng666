## What I build

<sub>Five products I build and run myself: a news-search tool for AI assistants, a notification system for AI coding tools, a language that turns text into floor plans, an AI design studio for architects, and an AI resume writer.</sub>

<img src="/public/brands/server-google-news.svg" align="right" height="80" alt="Google News MCP Server" title="Google News MCP Server">

### [Google News MCP Server](https://github.com/ChanMeng666/server-google-news)

<sub>Open source (MIT) — solo author and maintainer.</sub>


Lets AI assistants like Claude search live Google News — sorted into topics, in 10 languages. One of the first tools built for Anthropic's Model Context Protocol: shipped 35 days after the standard launched, and it's now listed across 15+ MCP directories.

<br clear="all">

<img src="/public/brands/echook-logo.svg" align="right" height="80" alt="echook (Echo + Hook)" title="echook (Echo + Hook)">

### [echook (Echo + Hook)](https://github.com/ChanMeng666/echook)

<sub>For <a href="https://claude.com/claude-code"><img src="/public/tech-logos/claude.svg" height="22" alt="Claude Code" style="vertical-align:middle"> Claude Code</a> · <a href="https://cursor.com"><img src="/public/tech-logos/cursor.svg" height="22" alt="Cursor" style="vertical-align:middle"> Cursor</a> · <a href="https://github.com/openai/codex"><img src="/public/tech-logos/codex.svg" height="22" alt="OpenAI Codex" style="vertical-align:middle"> OpenAI Codex</a>.</sub>


A notification system for AI coding assistants — Claude Code, Cursor, and Codex — that you install and set up just by talking to the AI in plain English. It plays sounds, reads the agent's reply aloud, warns you before you hit usage limits, and shows a live status bar.

<br clear="all">

<img src="/public/brands/archlang-logo.svg" align="right" height="80" alt="ArchLang" title="ArchLang">

### [ArchLang](https://playground.archlang.uk)



Describe a floor plan in plain text — walls, rooms, doors, windows — and get a precise, dimensioned architectural drawing back. Change one number and exactly one thing moves. It also checks that a plan is actually livable, and powers ArchCanvas, an AI design agent.

<br clear="all">

<img src="/public/brands/archcanvas-logo.svg" align="right" height="80" alt="ArchCanvas" title="ArchCanvas">

### [ArchCanvas](https://archcanvas.uk/)



An AI design agent for architects and self-builders: describe a building in plain words and it designs a dimensioned floor plan you can actually build from, plus a realistic rendering — then you refine it by talking, on an infinite zoomable canvas.

<br clear="all">

<img src="/public/brands/vitex.svg" align="right" height="80" alt="Vitex — AI Career Agent (vitex.org.nz)" title="Vitex — AI Career Agent (vitex.org.nz)">

### [Vitex](https://www.vitex.org.nz/)



Paste a job ad and describe your background — it writes a resume and cover letter tailored to that job and builds them into a polished PDF in about 30 seconds. Works from a browser, the command line, or inside an AI assistant.
