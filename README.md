<sub>For <a href="https://claude.com/claude-code"><img src="/public/tech-logos/claude.svg" height="22" alt="Claude Code" style="vertical-align:middle"> Claude Code</a> · <a href="https://cursor.com"><img src="/public/tech-logos/cursor.svg" height="22" alt="Cursor" style="vertical-align:middle"> Cursor</a> · <a href="https://github.com/openai/codex"><img src="/public/tech-logos/codex.svg" height="22" alt="OpenAI Codex" style="vertical-align:middle"> OpenAI Codex</a>.</sub>

### [ArchLang](https://archlang-playground.vercel.app)



An original declarative floor-plan language Chan invented — compiling to professional dimensioned plans (Typst/LaTeX for architecture). Zero-dependency isomorphic TypeScript with errors-as-data, soundness linting, and circulation modeling; the engine behind ArchCanvas.

<br clear="all">

<img src="/public/brands/archcanvas-logo.svg" align="right" height="80" alt="ArchCanvas" title="ArchCanvas">

### [ArchCanvas](https://archcanvas.chanmeng.org/)



An AI design agent for architects that productizes Chan's own ArchLang DSL — it writes parametric ArchLang (not freehand SVG) compiled to real dimensioned plans, with grounded GPT-Image renderings, a zoomable canvas, a review mode, and git-repo export. Solo-built, 269 commits.

### [Vitex](https://www.vitex.org.nz/)



Agent-ready AI career agent — an 8-step pipeline turns a JD + free-text background into a tailored ATS-optimised resume + cover letter in ~30s, usable over a public HTTP API, a CLI, and an MCP server; idempotent generation (committed only on compile), Typst PDF compile <100ms.
